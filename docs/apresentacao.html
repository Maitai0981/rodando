Resumo executivo
O Rodando Moto Center é uma plataforma completa de e-commerce para peças de motocicletas, desenvolvida com React no frontend e Spring Boot no backend. O sistema cobre o ciclo completo de vendas — catálogo, carrinho, checkout, pagamento e gestão de pedidos — além de um painel administrativo robusto para o proprietário. A aplicação está funcional, com testes automatizados passando e arquitetura preparada para deploy em produção.
